# Tools

## Tower authoring validation

Use `TWTD/Validation/Validate Tower Authoring` in the Unity top menu to validate:

- `TowerUpgradeTree` index links, reachability, and slot labels.
- `TowerEvolutionProfile` sprite/controller setup and `ShootArrow` animation event in fire clips.

Context menu shortcuts are available in Inspector:

- `TowerUpgradeTree -> Validate Tree`
- `TowerEvolutionProfile -> Validate Evolution Profile`
